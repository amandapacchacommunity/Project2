# Synthetic Enterprise Risk Register

This project recreates an enterprise risk register using fully synthetic data for portfolio and GitHub demonstration purposes.

## Project Overview
The repository includes a sample risk dataset and a basic Python script for summarizing risks by category.

## Files
- `data/synthetic_risk_register.csv` — synthetic risk register dataset
- `src/generate_data.py` — simple Python analysis script
- `README.md` — project documentation

## Example Use Case
This project demonstrates how risk data can be organized and analyzed for operational, financial, compliance, and strategic reporting.

## Skills Demonstrated
- Python
- Pandas
- Data analysis
- Risk reporting
- GitHub project organization

## Note
All data in this repository is synthetic and created for demonstration purposes only.
