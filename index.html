Community Risk Intelligence Dashboard

This project demonstrates how open data and analytics can be used to identify community risk patterns and support better decision making.

About the Project

The dashboard analyzes simulated risk register data and presents insights through interactive visualizations. The goal is to demonstrate how data science tools can support public sector planning, community programs, and risk monitoring.

Tools Used

  • Python
  • Streamlit
  • Pandas
  • Data Visualization
  • GitHub
